= openf4 =

== Description ==

openf4 is a c++ library for groebner basis computations over finite fields using the F4 algorithm.

* openmp (optional)
* givaro (optionnal)

== Special Update/Build Instructions ==

A C++ 11 compiler is required.
In order to have a multithread version, compile with g++ (clang does not support openmp).
